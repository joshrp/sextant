# Bugs & Problems

If something isn't working as expected, you can report it on the **GitHub Issues** page:

[**Open an Issue on GitHub**](https://github.com/joshrp/sextant)

When reporting a bug, it helps to include:

- **What you expected** to happen
- **What actually happened**
- **Steps to reproduce** the problem
- An **export string** of the factory if relevant (see [Import & Export](help?topic=import-export))

## Feature Requests

Have an idea for a new feature or improvement? Feature requests are welcome — open an issue on GitHub and describe what you'd like to see.

## Source Code

Sextant is open source. You can browse the code, fork the project, or contribute on GitHub
